# 🕌 Islamic Quiz App

A beautiful, mobile-friendly Islamic quiz web app built with HTML, CSS, and JavaScript.

## ✨ Features

- 20 Islamic questions (Quran, Hadith, History, Pillars of Islam)
- 15-second timer per question
- Live score tracker
- Beautiful Islamic theme (green & gold)
- Works on mobile and desktop
- Share your result with friends

## 🚀 How to Play

Just open the link — no installation needed!

👉 **Live Link:** `https://YOUR-USERNAME.github.io/islamic-quiz/`

- JavaScript (Vanilla)
